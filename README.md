# 🔌 Парсер графіків відключення світла Вінниці

Автоматичний парсер графіків відключення з voe.com.ua для Вінницької області.

**🚀 Запускається автоматично на GitHub Actions кожні 15 хвилин!**

## ⚡ Основні можливості

- ✅ Автоматичний парсинг графіків відключення Вінниці
- ✅ Запускається на серверах GitHub (без потреби у локальному хостингу)
- ✅ Підтримка Playwright для взаємодії з JS-контентом
- ✅ Структурований вивід у JSON формат
- ✅ Розпізнавання 7 типів станів світла
- ✅ Розподіл на поточний і наступний дні

## 📊 Формат JSON

```json
{
  "regionId": "vinnytsia",
  "lastUpdated": "2025-11-25T19:51:21.371Z",
  "fact": {
    "data": {
      "1764021600": {
        "GPV1.1": {"1": "yes", "2": "first", "3": "no"}
      }
    }
  }
}
```
