# 🔌 Парсер графіків відключення світла Вінниці

Автоматичний парсер графіків відключення з voe.com.ua для Вінницької області.

**🚀 Запускається автоматично на GitHub Actions кожні 15 хвилин!**

## ⚡ Основні можливості

- ✅ Автоматичний парсинг графіків відключення Вінниці
- ✅ Запускається на серверах GitHub
- ✅ Підтримка Playwright
- ✅ JSON вивід
